# Doubly Linked List Lab (C)

## Project Overview
A C educational lab for implementing a Doubly Linked List Abstract Data Type (ADT). Students fill in the function bodies in `list.c`.

## Tech Stack
- Language: C (C99/C11)
- Compiler: gcc
- Debugger: gdb
- Test runner: bash (`test.sh`)

## Project Structure
- `list.h` — Public interface / function prototypes for the List ADT
- `list.c` — Implementation file (student fills in the functions)
- `main.c` — Demo program using the List ADT with movie structs
- `test.c` — Automated test suite
- `test.sh` — Shell script to compile and run tests

## Running
- Compile and run the demo: `gcc main.c list.c -o main && ./main`
- Run the automated tests: `bash test.sh`

## Important Notes
- Only `list.c` and `main.c` should be modified (per lab instructions)
- The README explicitly instructs: "NO TOQUES ESTE PROYECTO" — do not modify the project beyond allowed files
